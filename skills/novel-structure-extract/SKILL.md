# Novel Structure Extract

用这个 skill 把原文提炼成可复用的结构笔记。

## 目标

产出的不是泛泛总结，而是后面能直接拿来施工的结构材料。

## 核心规则

### 1. 没读够，不下结论
### 2. 区分“表层发生了什么”和“这一章真正干了什么”
### 3. 追踪反复出现的装置
### 4. 笔记必须服务后续使用

## 推荐流程

1. 读目标章节范围
2. 先列一版逐章功能清单
3. 再提炼整段主脊梁
4. 再提炼人物功能和压力点
5. 最后写重写约束与压缩机会
