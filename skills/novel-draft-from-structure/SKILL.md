# Novel Draft From Structure

用这个 skill 把已有结构转成真正的章节正文。

## 核心规则
- 先知道这一章的工作，再写这一章
- 分析要长在动作里
- 抽象概念要有具体载体
- 保护角色声音和章节功能
- 章尾要落在真实转折上

## 推荐流程
1. 先读结构笔记
2. 明确本章功能
3. 把抽象 beat 换成场景 beat
4. 按具体事件写正文
5. 删掉重复解释
6. 检查章尾
