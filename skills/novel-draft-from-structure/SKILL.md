# Novel Draft From Structure

用这个 skill 把已有结构转成真正的章节正文。

## 用途

结构已经有了，接下来不是继续诊断，而是开始写小说。

如果任务来自长篇或超长篇，默认把“读原文”“做校准”“写正文”视为三个不同阶段；不要因为已经准备写正文，就假设前两步可以省略。

## 核心规则

### 1. 先知道这一章的工作，再写这一章

每一章在动笔前都应该明确：
- 这章改变了什么
- 这章把压力推到哪里
- 这章让什么信息变得可行动
- 为什么它值得单独成章

### 2. 分析要长在动作里

不要让人物站着把小说解释给彼此听，除非解释本身会立刻改变局势。

优先使用：
- 调查
- 冲突
- 执行任务
- 被打断的对话
- 物件驱动的发现
- 穿越空间时出现的新异常

### 3. 抽象概念要有具体载体

如果这一段涉及高概念，就把它落在：
- 物件
- 房间
- 工具
- 记录
- 伤口
- 仪式动作
- 名字与标签

### 4. 保护角色声音和章节功能

主角必须始终是主动思考的人。
配角不能沦为纯说明机器。

### 5. 章尾要落在真实转折上

好的章尾通常会做到其中之一：
- 重写问题本身
- 露出代价
- 改变目标
- 逼出选择
- 翻出更深一层威胁

## 推荐流程

1. 先读结构笔记
2. 先做续写校准
3. 明确这次续写的范围（补一章 / 补尾声 / 开新 arc）
4. 明确本章功能
5. 把抽象 beat 换成场景 beat
6. 按具体事件写正文
7. 删掉重复解释
8. 最后检查章尾是否真的成立

## 续写既有作品时，先做续写校准

默认顺序：
1. 先判断原结局/上一段已经解决了什么、没解决什么
2. 再看当前世界秩序和阶段状态
3. 再看主角、配角和关系网
4. 最后才决定第一章或第一段正文写什么

不要把这些步骤打乱成“边想边写”，否则容易凭惯性滑进不匹配的剧情。

如果任务不是“从零写一章”，而是接着已有小说、已有章节、已有结局继续写，动笔前先用要点形式写清以下内容，再开始正文：

### 1. 主角状态

- 主角当前的性格底色是什么
- 主角此阶段最核心的驱动力是什么
- 主角当前最怕失去什么、最在意什么
- 主角当前的社会位置是否已经变化（如从追赶者变成守成者、从边缘人物变成被注视者）
- 本章里主角说话和做决定，更接近他哪个时期、哪种状态

### 2. 关键配角与关系网

- 核心配角各自当前处于什么人生阶段
- 他们与主角的关系到了哪一步，是否已经发生性质变化
- 哪些配角仍然有叙事张力，哪些配角更适合退居背景
- 第一章里哪些配角必须出场，哪些适合延后出场，哪些只该被提及
- 配角在原作中的功能是什么，续写时不要随手改写成功能相反的人

### 3. 世界观与阶段秩序

- 当前世界观中，哪些规则是稳定成立的
- 最近的重大事件改变了什么力量格局、组织格局或日常秩序
- 哪些设定只是背景，哪些设定已经进入角色生活并产生压力
- 续写这一段时，世界是否应该扩大，还是应该先回到人物生活内部

### 4. 这一阶段适合写什么

- 这一阶段适合写什么冲突，不适合硬写什么冲突
- 原文叙事气质的关键词是什么（如日常/成长/关系/悬念/压迫感）
- 本章应该承接哪条已有压力，而不是临时制造一个不匹配的新主线

如果这些问题答不出来，先不要硬写正文，先补结构理解。

## 接在原作结局之后时，额外检查两件事

如果用户要“接着结局写”“继续写后续”“补结尾之后的故事”，不要直接顺着情绪往下写。

先额外确认：

- 原结局已经解决了什么主问题
- 原结局刻意没展开、但仍然留有张力的是什么

续写时优先承接“尚未展开但自然存在的压力”，不要随手制造一个与原作阶段不匹配的新主线。

## 人物与世界一致性保护

续写强角色、强世界观作品时，优先保护人物声音、关系阶段和世界秩序，而不是优先追求你自己的“合理剧情”。

语言层面不要只模仿表面口头禅或句式。优先模仿：
- 叙事节奏
- 信息密度
- 转折方式
- 日常与推进如何咬合

只在这些层面也对上时，表面风格才有意义。

判断标准：

- 这段话像不像这个角色会说的话
- 这个决定像不像这个阶段的他会做的决定
- 配角是否还在原本的功能和成长轨道上
- 世界对角色施加的限制和机会，是否仍然符合原文规则
- 情节推进是否保留了原文一贯的气质，而不是只保留设定名词

如果这些内容开始漂移，先停下来，回到“续写校准”。

## 先校准，后正文

如果任务是“接着已有作品继续写”，默认先产出一版简明的续写校准结果，再进入正文。

除非用户明确说“不要分析，直接写”，否则不要跳过这一步。

如果原文很长，而当前任务还要继续推进到正文，必要时把流程拆开：
- 第一步只做结构提炼或校准
- 第二步只决定续写范围和起手点
- 第三步才真正起稿

不要把计划、阅读、判断、正文一次性塞进同一次输出。
